# Tic Tac Toe

A simple two-player Tic Tac Toe game in Python, playable in the terminal.

## Overview
This project is a terminal-based Tic Tac Toe game where two players take turns marking X and O on a 3x3 grid. The game ends when one player gets three in a row (horizontally, vertically, or diagonally) or when all spots are filled, resulting in a draw.

## How to Play
1. Make sure you have Python installed.
2. Run the game in the terminal:
   ```bash
   python tic_tac_toe.py
